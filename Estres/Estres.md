## Que es el estrés

Se trata de una respuesta física y emocional ante situaciones que se sienten desafiantes o amenazantes, en algunas situaciones resulta útil, sin embargo cuando es algo constante puede afectar la salud física y mental.

## Como identificar el estrés
El primer paso para manejar el estrés es el identificar como se presenta, se puede ver cómo:

### Señales físicas
- Dolores de cabeza frecuentes.
- Tensión muscular (Generalmente en cuello, hombros y espalda).
- Fatiga o cansancio constante.
- Problemas para dormir.
- Aumento o disminución del apetito.
- Molestias estomacales.
### Señales emocionales
- Irritabilidad o enojo frecuente.
- Ansiedad o nerviosismo.
- Sentimientos de preocupación constante.
- Sensación de estar abrumado.
- Falta de motivación.

### Señales cognitivas 
- Dificultad para concentrarse.
- Problemas de memoria.
- Pensamientos negativos recurrentes.
- Dificultad para tomar decisiones.

### Señales conductuales
- Procrastinación.
- Aislamiento social.
- Uso desmedido de redes sociales o entretenimiento como escape.

# Manejo rápido del estrés
## Técnicas Rápidas (1-10 minutos)

### Respiración profunda
#### Pasos para hacerlo:
1. Inhala lentamente por la nariz durante 4 segundos.
2. Mantén el aire durante 4 segundos.
3. Exhala lentamente durante 6-8 segundos.
4. Repite varias veces 

Siguiendo estos pasos la frecuencia cardíaca y disminuye la sensación de ansiedad.

### Relajación muscular
#### Pasos para hacerlo:
1. Aprieta los músculos de una parte del cuerpo durante 5 segundos.
2. Relájalos completamente.
3. Repite con otras zonas.

Así la tensión acumulada se reducirá.  

### Movimiento breve
Para ello se tienen varias opciones, como:
- Caminar durante 5 minutos
- Estirarse.
- Subir y bajar escaleras.
Se libera la tensión física, mejorando el estado de ánimo.

### Escuchar música relajante
Esto reduce la alteración emocional y favorece la relajación rápida.
